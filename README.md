Overview

AI Interviewer is an intelligent system designed to conduct assessments, generate resumes and cover letters, and provide personalized feedback. It leverages AI to evaluate responses, suggest improvements, and enhance the interview and job application process.

Features

AI-Powered Assessments: Conduct interviews and technical assessments with AI-driven questions and evaluation.

Resume Builder: Generate professional resumes with AI assistance.

Cover Letter Generator: Create tailored cover letters based on job descriptions and user inputs.

Personalized AI Feedback: Receive AI-driven insights on assessments, resumes, and cover letters to improve job applications.

User Authentication: Secure login and profile management.

Data Storage: Save and manage resumes, cover letters, and interview history.

Technologies Used

Backend: Node.js, Express.js, MongoDB, Mongoose

Frontend: , React.js

AI & NLP: Gemini for text analysis and response evaluation

Authentication: Clerk Authentication
